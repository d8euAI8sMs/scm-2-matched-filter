Сборка из коробки работает из самой vs201x. Цели настроены таким образом, чтобы все работало автоматически.

## Инструменты

Для сборки проекта необходимы:

* `Visual Studio 2019 >= Community Edition`
* `rust >= 1.38.0`
* `cbindgen` (rust crate)
